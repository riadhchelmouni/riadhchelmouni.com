Project Overview

In the Push Swap Middle Sorter project, I developed an efficient sorting algorithm for the push_swap.c challenge at 42, designed to sort data using two stacks with a limited set of operations. The core of the algorithm involves dynamically selecting pivot elements and partitioning the data around these pivots, pushing elements to the appropriate stacks based on their comparison with the pivot. By dynamically selecting pivot points, the algorithm ensures an even distribution of elements, which are then divided into smaller subsets. These subsets are recursively sorted to guarantee the entire data set is sorted with minimal operations. The key to the algorithm's efficiency lies in its strategic pivot selection and effective stack management, which minimizes the number of operations required. This project enhanced my understanding of algorithms and data structures, and honed my skills in performance optimization and problem-solving. The resulting algorithm is highly efficient, demonstrating my ability to tackle complex coding challenges and develop optimized solutions for real-world applications.
